About Simulation Creation

Simulation Creation allows you to run new simulations based on existing Proforma schedules.

  • Select a proforma with trade and lane information
  • The simulation will use capacity and quantity from the selected proforma
  • Add an optional description for reference
  • The simulation will be executed in the background

Proforma details are displayed only. Port-specific information (berth windows) will be retrieved during simulation execution.
